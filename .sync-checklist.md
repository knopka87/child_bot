# Environment Sync Checklist

## После изменений на сервере всегда синхронизируйте:

### 1. Backend Environment
```bash
# Скачать с сервера
ssh -i ~/.ssh/id_rsa root@77.222.60.149 "cat /root/child_bot/.env" > .env.production.server

# Сравнить и обновить локальный
diff .env.production .env.production.server
# Если есть изменения - обновить .env.production вручную
```

### 2. Frontend Environment
```bash
# Локальный файл
cat frontend/.env.production

# Ключевые переменные:
# - VITE_API_BASE_URL=/api  (для production)
```

### 3. Docker Compose
```bash
# Если изменился docker-compose.yml на сервере
ssh -i ~/.ssh/id_rsa root@77.222.60.149 "cat /root/child_bot/docker-compose.yml" > docker-compose.production.yml.server

# Сравнить
diff docker-compose.production.yml docker-compose.production.yml.server
```

## Критические переменные для синхронизации

### Backend (.env.production)
- `POSTGRES_PASSWORD` - пароль БД
- `REDIS_PASSWORD` - пароль Redis
- `VK_APP_SECRET` - секрет VK приложения
- `EMAIL_API_KEY` - ключ email сервиса

### Frontend (frontend/.env.production)
- `VITE_API_BASE_URL` - базовый URL API (должен быть `/` для production)
- `VITE_VK_APP_ID` - ID VK приложения

## Quick Sync Command
```bash
# Полная синхронизация с сервера
ssh -i ~/.ssh/id_rsa root@77.222.60.149 "cat /root/child_bot/.env" > /tmp/server.env
echo "Compare and update manually:"
echo "diff .env.production /tmp/server.env"
```
