### 3. Улучшены фильтры

**Было:**
```
[Все] [Помощь] [Проверка] | [Все статусы] [Верно] [Ошибки]
```

**Стало:**
```
Режим:
  [Все] [Помощь] [Проверка]

Результат:
  [Все] [Верно] [Ошибки]
```

**Объяснение разницы:**
- **Режим** (Все/Помощь/Проверка) - фильтр по **типу попытки** (help или check)
- **Результат** (Все/Верно/Ошибки) - фильтр по **статусу выполнения** (success или error)

Это **разные фильтры**, поэтому оставлены оба с явным разделением и подписями.

---

### 4. Новые статусы

| Статус | Приоритет | Иконка | Цвет | Текст |
|--------|-----------|--------|------|-------|
| hints | 1 (высший) | 💡 Lightbulb | Фиолетовый | "Использованы подсказки" |
| success | 2 | ✓ CheckCircle | Зелёный | "Решено верно" |
| error | 3 | ✗ XCircle | Красный | "Есть ошибки" |
| in_progress | 4 | ⚙ Loader | Серый | "В обработке" |
| default | 5 | ⏱ Clock | Жёлтый | "Незакончено" |

**Логика приоритета:**
- Если `hintsUsed > 0` → показываем "Использованы подсказки" (даже если решено верно)
- Иначе показываем статус по результату
