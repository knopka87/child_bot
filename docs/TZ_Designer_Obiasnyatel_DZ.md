## ФАЗА 0: Quick Wins в Telegram-боте

> **Цель**: Получить WOW-эффект БЕЗ разработки Mini App. Только тексты, эмодзи, стикеры и GIF в обычном Telegram.

### 0.1 Streak-механика (серия дней) 🔥

**Что нужно от дизайнера:**
* **Набор стикеров/PNG** для разных уровней streak:
  - 🔥 1-2 дня: маленький огонёк
  - 🔥🔥 3-6 дней: средний огонь
  - 🔥🔥🔥 7+ дней: большое пламя, искры
  - 💔 Потеря streak: грустный маскот с потухшим огоньком
* **Milestone-стикеры**: 7 дней, 14 дней, 30 дней (особые)

**Примеры сообщений бота:**
```
🔥 Серия: 5 дней подряд!
Ещё 2 дня до нового рекорда 💪

---

💔 Ой, серия прервалась...
Но ничего! Начнём новую прямо сейчас 🚀
```

### 0.2 Система достижений (Badges)

**Что нужно от дизайнера:**
* **15-20 стикеров достижений** в едином стиле:

| Достижение | Условие | Визуал |
|------------|---------|--------|
| 🌟 Первые шаги | 1 задача | Звёздочка |
| 🎯 Меткий стрелок | 5 задач | Мишень |
| 🧠 Умник | 10 задач | Мозг/лампочка |
| 🚀 Ракета | 25 задач | Ракета |
| 🏆 Чемпион | 50 задач | Кубок |
| 🔥 Огонь! | 7 дней streak | Пламя |
| 🦉 Сова | Решил задачу после 21:00 | Сова |
| 🌅 Ранняя пташка | Решил до 8:00 | Солнце |
| ⚡ Молния | 3 задачи за день | Молния |
| 🎓 Профессор | Все 3 подсказки не понадобились | Шапка выпускника |

### 0.3 Эмоциональные реакции бота

**Что нужно от дизайнера:**
* **Набор GIF/стикеров маскота** для разных ситуаций:
  - Thinking (думает, крутит лупу)
  - Celebrating (прыгает, confetti)
  - Encouraging (подбадривает, показывает "класс")
  - Proud (гордится, улыбается)
  - Oops (грустит, но не сдаётся)

**Примеры живых реплик вместо сухих:**
```
❌ Было: "Подсказка 1: ..."
✅ Стало: "💡 Лови первую подсказку! [стикер думающего маскота]"

❌ Было: "Ответ верный"
✅ Стало: "🎉 БДЫЩ! Ты справился! [GIF празднующего маскота]"
```

### 0.4 Дневная статистика для родителя

**Формат сообщения (текст + эмодзи):**
```
📊 Отчёт за сегодня

✅ Решено задач: 3
📚 Темы: умножение, задачи на движение
⏱ Время занятий: ~15 мин
🔥 Серия: 5 дней подряд!
🏅 Новое достижение: "Меткий стрелок"

Так держать! 💪
```

**Что нужно от дизайнера:**
* Шаблон карточки-отчёта (если решим делать как изображение)

## 1. Общие требования к визуалу

* **Стиль**: «Умный напарник» (vibe 12+), современный и технологичный. Избегать излишней «мультяшности».
* **Цвета**:
  - Основной: синий (Лидер, #3B82F6)
  - Акцентный: жёлтый (Энергия, #FBBF24)
  - Успех: зелёный (#10B981)
  - Ошибка: красный (#EF4444)
* **Форматы**:
  - Фаза 0: PNG/WebP стикеры, GIF
  - Фаза 1+: Lottie (JSON) для анимаций

## ФАЗА 1: Mini App MVP

> **Цель**: Базовая геймификация с визуальным WOW-эффектом

### 2. Навигация (Tab Bar) `[Фаза 1]`

Начальная версия — 4 вкладки:

| # | Вкладка | Иконка | Контент |
|---|---------|--------|---------|
| 1 | **Штаб** | 🏠 | Маскот, streak, питомец, босс недели |
| 2 | **История** | 📋 | Архив задач |
| 3 | **Коллекция** | 🏆 | Достижения, трофеи |
| 4 | **Родителям** | 👨‍👩‍👧 | Parental Gate → Портал |

### 3. Спецификация экранов `[Фаза 1]`

#### 3.1 Главный экран (Штаб-квартира)
* Центр: анимированный Герой-Исследователь
* Сверху: Streak-индикатор 🔥
* Справа: Мини-виджет питомца
* Снизу: Прогресс босса недели
* Кнопки: «Помоги решить» (💡) и «Проверишь меня?» (✅)

#### 3.2 Экран сканера
* Полноэкранный режим камеры
* Анимированная лазерная линия сканирования
* Рамки-подсветки найденных элементов

#### 3.3 Экран результата (Hints)
* Карточка с текстом задачи
* Три блока подсказок (L1/L2/L3) — раскрываются по очереди
* Эффекты: «БДЫЩ!» и «ЭВРИКА!» при успехе

#### 3.4 Экран верификации
* Сравнение фото решения с эталоном
* Красные/зелёные маркеры ошибок
* «Грустный монстр» при неудаче

### 4. Питомец-компаньон `[Фаза 1 — MVP]`

> **Ключевая фича для эмоциональной привязки**

**Базовая версия (MVP):**
* **3 состояния**: Голодный → Сытый → Счастливый
* **1 вид питомца** на старте (например, дракончик)
* **Простая анимация** кормления

**Элементы для дизайна:**
* Виджет питомца (150x150px) для главного экрана
* 3 состояния в Lottie:
  - Голодный: грустные глаза, серый оттенок, медленное покачивание
  - Сытый: нормальный цвет, моргание
  - Счастливый: светится, прыгает, звёздочки вокруг
* Анимация кормления: звёздочка летит к питомцу, он "ест"

### 5. Босс недели `[Фаза 1]`

**Элементы для дизайна:**
* **3 разных босса** на ротацию:
  - Дракон Забывания (синий)
  - Король Ошибок (красный)
  - Туман Лени (серый)
* **Полоска HP** босса (уменьшается с задачами)
* **Состояния**: Полный HP → Ослаблен (50%) → Побеждён
* **Мини-виджет** для главного экрана (100x60px)
* **Анимация атаки**: вспышка при решении задачи
* **Анимация победы**: босс рассыпается, confetti

### 6. Кнопка "Похвалить" `[Фаза 1]`

**В родительском портале:**
* Кнопка "Отправить похвалу"
* Выбор из 5 стикеров: "Молодец!", "Горжусь!", "Так держать!", "Супер!", "Люблю!"
* Опционально: кастомный текст

**У ребёнка:**
* Push-уведомление
* Всплывающее сообщение с анимацией сердечек
* Сохранение в "Архив похвал"

### 7. Комната трофеев `[Фаза 1]`

**Упрощённая версия (не 3D):**
* Сетка/список достижений
* Полученные — цветные, с датой
* Не полученные — силуэты с условием
* Редкость: обычные / редкие (свечение) / легендарные

## ФАЗА 2: Расширение

### 8. Вовлечение и Retention `[Фаза 2]`

#### 8.1 Миссия дня (Daily Challenge)
* Карточка миссии на главном экране
* Таймер до новой миссии
* Награда за выполнение

#### 8.2 Эволюция питомца
* 4 стадии: яйцо → малыш → подросток → взрослый
* Выбор из 4 видов питомцев

#### 8.3 Кастомизация маскота
* Гардероб: шапки, очки, аксессуары
* Источники: награды, магазин за очки

### 9. Социальное взаимодействие `[Фаза 2]`

#### 9.1 Режим "Помоги другу"
* Отправить задачу другу
* Входящие просьбы о помощи
* Награда "Лучший помощник"

#### 9.2 Таблица лидеров
* Топ-10 класса (опционально, включается родителем)
* Фильтры: день / неделя / всё время

### 10. Повторение и закрепление `[Фаза 2]`

#### 10.1 Машина времени (Spaced Repetition)
* Возврат к старым задачам
* Календарь повторений

#### 10.2 Карта знаний ученика
* Дерево тем: серый → жёлтый → зелёный → золотой
* Анимация "расцветания" при освоении

### 11. Семья и родители `[Фаза 2]`

#### 11.1 Родительский квест (Family Quest)
* Совместные задания
* Семейные достижения

#### 11.2 Расширение родительского портала
* График прогресса по неделям
* Рекомендации "Обратите внимание на тему X"
* Экспорт PDF для учителя

## ФАЗА 3: Продвинутые фичи

### 12. Мини-игры (Brain Breaks) `[Фаза 3]`

* Поймай числа (падающие цифры)
* Собери слово (drag-and-drop)
* Найди пару (карточки памяти)
* Лопни пузыри (правильные ответы)

### 13. Голосовой режим `[Фаза 3]`

* Кнопка микрофона на экране подсказок
* Визуализация звуковой волны
* Транскрипция в облачке
* Маскот в состоянии "Слушаю"

### 14. Сезонные события `[Фаза 3]`

* Новогодний колпак, школьный ранец (1 сентября)
* Тематические фоны
* Ограниченные награды "Limited"

## Технические требования

### Форматы по фазам:
* **Фаза 0**: PNG (512x512 для стикеров), GIF (до 256KB)
* **Фаза 1+**: Lottie JSON, WebP

### Состояния UI-элементов:
* Active
* Disabled / Coming Soon
* Loading (скелетоны)
* Empty state

### Темы:
* Светлая (по умолчанию)
* Тёмная (наследуется от Telegram)

### Haptics (тактильная обратная связь):
* Успех: короткая вибрация
* Streak milestone: двойная вибрация
* Ошибка: лёгкая вибрация
* Получение награды: длинная вибрация
